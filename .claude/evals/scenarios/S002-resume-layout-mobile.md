# S002: Resume layout — mobile

## Preconditions
- Dev server running on localhost:3000

## Steps
1. Navigate to http://localhost:3000 at 375x667 viewport
2. Take a full-page screenshot
3. Verify nav: logo visible, nav links HIDDEN, chat button visible
4. Verify hero: single column, photo above text, floating cards hidden
5. Verify metrics strip: 2-column layout
6. Verify strength cards: 1 per row
7. Verify experience items: single column, meta not sticky
8. Verify portfolio: 1 card per row
9. Verify skills: 1 group per row

## Expected Results
- No horizontal overflow at 375px
- All touch targets at least 44x44px
- Text readable without zoom
- Single-column layout for all grids
