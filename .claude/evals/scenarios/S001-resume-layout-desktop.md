# S001: Resume layout — desktop

## Preconditions
- Dev server running on localhost:3000

## Steps
1. Navigate to http://localhost:3000 at 1280x800 viewport
2. Take a full-page screenshot
3. Verify nav is fixed at top with logo, nav links, and "Talk to My AI Clone" button
4. Verify hero section: photo on right, text on left, floating cards visible
5. Verify metrics strip (dark background, 5 metrics)
6. Scroll down — verify sections appear in order: Strengths, Experience, Portfolio, Skills, Education, CTA, Footer
7. Verify scroll reveal animations trigger as sections enter viewport

## Expected Results
- Layout matches mockup in mockup/fernando_ott_resume_v2.html
- Nav is sticky (stays visible on scroll)
- All 6 main sections visible
- No horizontal overflow
- Fonts load correctly (Instrument Serif for headings, Satoshi for body)
